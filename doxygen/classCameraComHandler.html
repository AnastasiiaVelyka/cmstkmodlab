◆ acquireAndDownloadPicture()

bool CameraComHandler::acquireAndDownloadPicture ( int  filedescriptor) const

Instructs the camera to take a picture, downloads it and writes it in the file given by filedescriptor. In order to do this, the file needs to have at least write permissions for the current user. After acquisition the file will be deleted from the camera.

  • filedescriptor File descriptor of a file that is writable by the current user.
    Returns
    true if picture was acquired and saved, false otherwise.
