Detailed Description

Command and control model of the Julabo chiller.

Constructor & Destructor Documentation

◆ JulaboModel()

JulaboModel::JulaboModel ( float  updateInterval = 5,
QObject *  parent = 0 
)
explicit

Creates a new JulaboModel for command and control of the chiller. The JulaboModel periodically checks for updated read-outs with the Julabo FP50 chiller used to cool the experimental setup. The default update interval is 5 seconds. The allowed ranges for the chiller parameters are hardcoded in the Parameter-object fields using the values provided by the Julabo manual.

  • updateInterval Time between read-out cache refreshes; in seconds.
  • parent Pointer to the parent QObject.
